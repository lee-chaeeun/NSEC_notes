            twee

Locate files in Linux

  • which : searches directories defined in $PATH environment variable for given file name
  • locate: quickest command to find file location
  • locate searches built-in DB, locate.db
  • Update DB using updatedb
  • find: difficult/flexible command to locate file. 
    
        └─$ echo $PATH
        /usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:

        └─$ which sbd
        /usr/bin/sbd

        └─$ sudo updatedb

        └─$ locate sbd.exe
        /usr/share/windows-resources/sbd/sbd.exe

        └─$ sudo find / -name sbd*
        /var/lib/dpkg/info/sbd.md5sums
        /var/lib/dpkg/info/sbd.list
        /usr/bin/sbd
        /usr/share/windows-resources/sbd
        /usr/share/windows-resources/sbd/sbdbg.exe
        /usr/share/windows-resources/sbd/sbd.exe
        /usr/share/doc/sbd
Kali Services

Pre-installed features, such as SSH, HTTP, MySQL...

Security measures : network services do not start at boot time/ whitelist & blacklist

SSH : Secure SHell is a protocol to access a computer remotely in an encrypted and secure manner

  • TCP based protocl running on port 22 by default
  • ss : verify that SSH service running on port 22
  • grep : used to pipe out only output of sshd
  • enable : allows for automatic start of SSH at boot time
      
        └─$ sudo systemctl start ssh 
        └─$ sudo ss -antlp | grep sshd
        LISTEN 0      128          0.0.0.0:22        0.0.0.0:*    users:(("sshd",pid=12345,fd=3))
        LISTEN 0      128             [::]:22           [::]:*    users:(("sshd",pid=12345,fd=4))
        └─$ sudo systemctl enable ssh
        └─$ sudo systemctl stop ssh
